# OBS Effects

A comprehensive collection of animated OBS backgrounds, overlays, and camera frames built with PixiJS and WebGL.

## Features

- **Animated Backgrounds**: Galaxy, nebula, particle effects, plasma waves, and more
- **Camera Frames**: Animated borders and overlays for webcam feeds
- **Interactive Effects**: Matrix rain, cosmic portals, gravitational simulations, flight simulations
- **High Performance**: Optimized rendering with WebGL acceleration and PixiJS v8
- **Live Streaming Ready**: Perfect for Twitch, YouTube, and other streaming platforms

## Use Cases

- Live stream overlays and backgrounds
- OBS browser sources
- Web-based animation library
- Real-time visual effects demonstrations

## Technologies

- PixiJS v8 - WebGL rendering engine
- TypeScript - Type-safe development
- Vite - Fast build tool
- Docker - Easy deployment and distribution

Perfect for content creators, streamers, and developers looking for production-ready animated visual effects.
